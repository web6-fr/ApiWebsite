# StaticWebsite

My personal static website development environment based on Grunt.

## Features

- SASS (libsass)
- CSS Minification
- CSS prefixing
- CSS purification
- HTML Minification
- JS Concatenation / Minification
- EJS Templates
- Dev / Dist environments
- LiveReload
- XmlSitemap
- Image compression
